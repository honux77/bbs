# Honux Blog

- Hugo 기반 정적 블로그, GitHub Pages 호스팅
- 테마: [hugo.386k](https://github.com/honux77/hugo.386k) (레트로 BBS 스타일)

## 기능

- **페이지뷰 배지** - [Hits](https://hits.seeyoufarm.com) (포스트별 조회수)
- **방문자 통계** - [Goatcounter](https://goatcounter.com) (대시보드: honux-jung.goatcounter.com)
- **댓글** - [Giscus](https://giscus.app) (GitHub Discussions 기반)

## 설정 (config.toml)

| 파라미터 | 설명 |
|---|---|
| `hitsEnabled` | 페이지뷰 배지 표시 여부 |
| `goatcounterCode` | Goatcounter 사이트 코드 |
| `giscusRepo` | Giscus GitHub 저장소 |
| `giscusRepoId` | Giscus 저장소 ID |
| `giscusCategory` | Giscus Discussion 카테고리 |
| `giscusCategoryId` | Giscus 카테고리 ID |
| `giscusTheme` | Giscus 테마 |
